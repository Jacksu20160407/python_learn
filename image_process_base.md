# Image process
### Image Convolution
数学的卷积两个函数乘积的积分，其中一个函数需要反转并平移。
图像卷积是将卷积核旋转180°后与原图像对应位置相乘在求和，得到的结果作为原图像中与卷积核中心位置相对应的位置的像素值
边界处理一般采用padding 0的方法
